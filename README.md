# Классификация изображений одежды по стилю на основе собранных референсов.


## Цель исследования:
1. Построить модель для классификации изображений одежды по стилю на основе собранных референсов.

Данные получим из интернет-ресурса “Pinterest”, выберем 30 изображений стиля “Old money” и 30 изображений стиля “Grunge”, проведем анализ стиля.
После этого выгрузим датасет в Jupyter Notebook, просмотрим несколько изображений, проверим соответствие классам.
Затем приступим к обучению модели и анализу метрик.
Завершающим этапом исследования будет проверка на тестовой выборке.
